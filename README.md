OmegleValentineBot
==================

Usage:
------
* Goes on Omegle.
* Enters a few topics.
* Starts messaging.
* Back to Step 1

Technical Info:
---------------
* The code is written in Java.
* Selenium is used to interact with Omegle.
* JokesDataSet was extracted from a couple of webpages.
